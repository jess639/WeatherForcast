# WeatherForcast
一个天气预报小Demo，实现方式采用MVP的上一些开源组件实现

    compile 'me.yokeyword:indexablestickylistview:0.6.1'
    compile 'com.github.kanytu:android-parallax-recyclerview:v1.7'
    还有一个ApiStoreSDK1.0.4.jar获取天气数据的jar包
